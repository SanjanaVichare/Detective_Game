# 🕵️ Detective SQL Mystery Game

An interactive detective game where players solve a crime by querying a database and uncovering clues.

The game combines **SQL investigation mechanics with a web interface** to create a fun DBMS learning experience.

## 🚀 Features

* Interactive mystery investigation
* SQL-based clue discovery
* Web interface built with Vite + TypeScript
* Database-driven gameplay
* Puzzle-based progression

## 🧠 Concept

Players act as detectives investigating a crime.
Clues are hidden inside the database, and players must use logical reasoning and queries to uncover the truth.

## 🛠 Tech Stack

* TypeScript
* Vite
* Node.js
* SQLite
* Tailwind CSS

## 📂 Project Structure

src/ → Game logic and UI
database/ → SQL database and clues
public/ → Static assets
server.js → Backend server
detective.db → Game database

## ▶️ Run the Project

Install dependencies

npm install

Start the server

npm run dev
